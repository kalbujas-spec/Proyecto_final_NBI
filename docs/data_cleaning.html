Preparación y Limpieza de Datos

Introducción

En esta sección se documenta el proceso de limpieza, integración y transformación de los datos utilizados para el análisis del Índice de Necesidades Básicas Insatisfechas (NBI).

El objetivo es garantizar que los datos estén consistentes, completos y listos para la etapa de visualización.

Diagrama del proceso del dataset

Fuente de datos

La base de datos utilizada proviene del Censo Nacional de Población y Vivienda 2022 del INEC.

Dimensiones del Índice NBI

A continuación, se describen las dimensiones que componen el indicador. Selecciona cada pestaña para ver el detalle:

  • 💰 Capacidad económica
  • 🎓 Acceso a educación básica
  • 🏠 Acceso a vivienda
  • 🚿 Acceso a servicios básicos
  • 👨‍👩‍👧 Hacinamiento

El hogar se considera privado en esta dimensión si se cumplen ambas condiciones:

  • Los años de escolaridad del jefe(a) de hogar son menores o iguales a 2 años
  • Existen más de tres personas por cada persona ocupada

Esta dimensión refleja limitaciones en capital humano y alta dependencia económica.

El hogar se considera privado si:

  • Existen niños de 6 a 12 años que no asisten a clases

Esta dimensión mide exclusión educativa en edades clave.

Se considera privación si se cumple al menos una de las siguientes condiciones:

  • Piso de tierra u otros materiales precarios
  • Paredes de caña, estera u otros materiales inadecuados

Evalúa la calidad estructural de la vivienda.

Se considera privación si:

  • No existe servicio higiénico adecuado
  • El agua no proviene de una fuente segura

Refleja condiciones sanitarias del hogar.

Se considera privación si:

  • Hay más de 3 personas por dormitorio

Indica sobreocupación del hogar.

Interpretación del NBI

Un hogar se clasifica como pobre por NBI si presenta al menos una de las cinco privaciones.

Este enfoque permite medir la pobreza desde múltiples dimensiones, superando las limitaciones de indicadores basados únicamente en ingresos.

Contexto de los datos

Debido a la magnitud del conjunto de datos original, el análisis se realizó únicamente sobre las variables de interés, garantizando que el informe sea estable, renderizable y reproducible.

Transformaciones realizadas

Agregación por provincia

Los datos individuales (i) fueron resumidos a nivel de provincia (p):

\[ X_{p} = \sum_{i \in p} x_i \]

Cuando corresponde, se calcularon promedios:

\[ \bar{X}_{p} = \frac{1}{N_p} \sum_{i \in p} x_i \]

Desagregación por área (urbana/rural)

Para cada provincia (p) y área (a {, }):

\[ X_{p,a} = \sum_{i \in (p,a)} x_i \]

Tablas colapsadas

Las variables fueron consolidadas mediante operaciones de colapso:

\[ \text{collapse}_{\text{provincia}} \left( \text{mean}(x),\ \text{sum}(y),\ \text{count}(n) \right) \]
